# 中华人民共和国社会保险法基础知识

## 一、社会保险概述

### 1.1 社会保险的定义
社会保险是国家通过立法强制实行的，由用人单位和劳动者共同缴纳，在劳动者因年老、患病、工伤、失业、生育等情况减少劳动收入时，给予经济补偿，使其能够享有基本生活保障的一项社会保障制度。

### 1.2 社会保险的五大险种
1. **基本养老保险**：保障劳动者退休后的基本生活
2. **基本医疗保险**：保障劳动者因病或非因工负伤时的基本医疗需求
3. **工伤保险**：保障劳动者因工作原因遭受事故伤害或患职业病时的医疗救治和经济补偿
4. **失业保险**：保障劳动者失业期间的基本生活
5. **生育保险**：保障女性职工生育期间的医疗费用和生育津贴

## 二、基本养老保险

### 2.1 参保范围
- 企业职工应当参加基本养老保险
- 无雇工的个体工商户、未在用人单位参加基本养老保险的非全日制从业人员以及其他灵活就业人员可以参加基本养老保险

### 2.2 缴费比例（2024年标准）
- **单位缴费比例**：一般为工资总额的16%
- **个人缴费比例**：一般为本人缴费工资的8%

### 2.3 养老金领取条件
- 达到法定退休年龄
- 累计缴费满最低年限（2029年底前为15年；2030年起逐步提高至20年，每年提高6个月）

### 2.4 法定退休年龄（2025年起渐进式延迟）

根据2024年9月13日全国人大常委会通过的《关于实施渐进式延迟法定退休年龄的决定》，自2025年1月1日起实施渐进式延迟退休。

**延迟目标：**
- **男性**：从60周岁逐步延迟至63周岁（每4个月延迟1个月）
- **女性干部**：从55周岁逐步延迟至58周岁（每4个月延迟1个月）
- **女性工人**：从50周岁逐步延迟至55周岁（每2个月延迟1个月）

**15年过渡期**，从2025年持续至2039年12月。

**最低缴费年限调整：**
- 2029年底前退休的，最低缴费年限仍为15年；
- 从2030年1月1日起，最低缴费年限由15年逐步提高至20年（每年提高6个月，至2039年达到20年）。

**弹性退休制度：**
- 达到最低缴费年限后，可在法定退休年龄基础上自愿提前退休（最多提前3年，且不低于原法定退休年龄）；
- 经与用人单位协商一致，也可选择延迟退休（最多延迟3年）。

## 三、基本医疗保险

### 3.1 参保范围
- 所有企业、机关事业单位、社会团体、民办非企业单位及其职工都应参加基本医疗保险
- 城镇居民可参加城镇居民基本医疗保险
- 农村居民可参加新型农村合作医疗

### 3.2 缴费比例（2024年标准）
- **单位缴费比例**：一般为工资总额的6%-10%（各地不同）
- **个人缴费比例**：一般为本人缴费工资的2%

### 3.3 医疗保险待遇
- 门诊医疗费用报销
- 住院医疗费用报销
- 特殊病种门诊治疗费用报销
- 大病医疗互助

### 3.4 医保个人账户（门诊共济改革后）

根据2021年国务院办公厅《关于建立健全职工基本医疗保险门诊共济保障机制的指导意见》：

**个人账户计入方式改革：**
- 个人缴纳的基本医疗保险费（本人缴费工资的2%）仍全部计入个人账户；
- **单位缴纳的部分不再划入个人账户**，全部计入统筹基金（改革前约30%划入个人账户）。

**门诊共济保障：**
- 普通门诊医疗费用纳入统筹基金报销范围，报销比例从50%起步；
- 高血压、糖尿病等慢性病门诊用药纳入统筹报销。

**个人账户使用范围扩大（家庭共济）：**
- 个人账户资金可用于支付**本人及其配偶、父母、子女**在定点医疗机构就医发生的由个人负担的医疗费用；
- 可用于在定点零售药店购买药品、医疗器械、医用耗材的个人负担费用；
- 可用于配偶、父母、子女参加城乡居民基本医疗保险的个人缴费。

## 四、工伤保险

### 4.1 参保范围
所有企业、事业单位、社会团体、民办非企业单位、基金会、律师事务所、会计师事务所等组织和有雇工的个体工商户都应当参加工伤保险。

### 4.2 工伤认定情形
- 在工作时间和工作场所内，因工作原因受到事故伤害的
- 工作时间前后在工作场所内，从事与工作有关的预备性或者收尾性工作受到事故伤害的
- 在工作时间和工作场所内，因履行工作职责受到暴力等意外伤害的
- 患职业病的
- 因工外出期间，由于工作原因受到伤害或者发生事故下落不明的
- 在上下班途中，受到非本人主要责任的交通事故或者城市轨道交通、客运轮渡、火车事故伤害的

### 4.3 工伤待遇
- **医疗待遇**：治疗工伤所需费用符合工伤保险诊疗项目目录、工伤保险药品目录、工伤保险住院服务标准的，从工伤保险基金支付
- **停工留薪期**：一般不超过12个月，最长不超过24个月
- **伤残补助金**：根据伤残等级不同，一次性支付相应金额
- **工亡待遇**：丧葬补助金、供养亲属抚恤金、一次性工亡补助金

### 4.4 工伤认定申请期限
- 用人单位应当在事故伤害发生之日或者被诊断、鉴定为职业病之日起30日内提出工伤认定申请
- 遇有特殊情况，经报社会保险行政部门同意，可以适当延长
- 单位未按时申请的，工伤职工或者其近亲属、工会组织在1年内可以直接提出工伤认定申请

## 五、失业保险

### 5.1 参保范围
城镇企业事业单位、城镇企业事业单位职工依照规定缴纳失业保险费。

### 5.2 缴费比例
- **单位缴费比例**：一般为工资总额的0.5%-1%
- **个人缴费比例**：一般为本人缴费工资的0.2%-0.5%
- **注意**：各地多执行阶段性降低费率政策，实际比例可能低于上述标准，以当地最新政策为准。

### 5.3 失业金领取条件
- 失业前用人单位和本人已经缴纳失业保险费满一年
- 非因本人意愿中断就业的
- 已经进行失业登记，并有求职要求的

### 5.4 失业金领取标准和期限
- **领取标准**：低于当地最低工资标准，高于城市居民最低生活保障标准
- **领取期限**：
  - 累计缴费满1年不足5年的：最长12个月
  - 累计缴费满5年不足10年的：最长18个月
  - 累计缴费10年以上的：最长24个月

## 六、生育保险（已与医疗保险合并实施）

> **政策更新**：根据国务院办公厅《关于全面推进生育保险和职工基本医疗保险合并实施的意见》（国办发〔2019〕10号），自2019年起生育保险与职工基本医疗保险**合并实施**，统一参保登记、统一基金征缴和管理、统一医疗服务管理、统一经办和信息服务。职工生育期间的生育保险待遇不变。

### 6.1 参保范围
参加职工基本医疗保险的在职职工同步参加生育保险，个人不缴纳生育保险费（由单位缴纳）。

### 6.2 生育保险待遇
- **生育医疗费用**：包括检查费、接生费、手术费、住院费和药费等
- **产假津贴**：按照用人单位上年度职工月平均工资计发
  - 正常产假98天
  - 难产增加15天
  - 多胞胎每多一个婴儿增加15天
  - 流产假期：怀孕不满4个月的15天；怀孕4个月以上的42天

### 6.3 男性职工生育保险权益
- 配偶生育时，可享受陪产假（各地规定不同，通常为7-30天）
- 未就业配偶可享受部分生育医疗费用待遇

## 七、社会保险征缴与监管

### 7.1 缴费基数确定
- 职工工资收入高于当地上年度职工平均工资300%的，以当地上年度职工平均工资的300%为缴费基数
- 职工工资收入低于当地上年度职工平均工资60%的，以当地上年度职工平均工资的60%为缴费基数

### 7.2 用人单位的义务
- 自用工之日起三十日内为其职工向社会保险经办机构申请办理社会保险登记
- 按时足额缴纳社会保险费
- 代扣代缴职工个人应缴纳的社会保险费
- 向职工告知社会保险参保情况和缴费情况

### 7.3 违法责任
- 用人单位未按规定办理社会保险登记的，由社会保险行政部门责令限期改正
- 逾期不改正的，对用人单位处应缴社会保险费数额一倍以上三倍以下的罚款
- 对其直接负责的主管人员和其他直接责任人员处五百元以上三千元以下的罚款

## 八、社会保险争议处理

### 8.1 争议类型
- 用人单位不为员工缴纳社会保险费的
- 缴费基数不实的
- 不按规定享受社会保险待遇的

### 8.2 解决途径
1. **协商**：与用人单位协商解决
2. **投诉举报**：向社保行政部门投诉举报
3. **劳动仲裁**：向劳动仲裁委员会申请仲裁
4. **行政诉讼**：对社保行政部门的处理决定不服的，可以提起行政复议或行政诉讼

### 8.3 时效规定
- 劳动仲裁时效为一年，从当事人知道或者应当知道其权利被侵害之日起计算
- 社保补缴不受仲裁时效限制，但受追诉时效限制

## 九、社会保险关系转移接续

### 9.1 养老保险关系转移
1. **跨省流动就业**：参保人员跨省流动就业的，其基本养老保险关系随同转移，个人账户储存额全部转移，统筹基金按12%的比例转移。
2. **办理渠道**：可通过"国家社会保险公共服务平台"、"掌上12333"APP或各地社保经办机构申请办理。
3. **转移条件**：在新参保地建立基本养老保险关系并缴费后，可申请将原参保地养老保险关系转入。
4. **待遇领取地确定**（"户籍优先、从长从后"原则）：
   - 基本养老保险关系在户籍所在地的，在户籍所在地领取；
   - 不在户籍所在地，但在现参保地累计缴费满10年的，在现参保地领取；
   - 不在户籍所在地且在每个参保地累计缴费均不满10年的，将关系归集至户籍所在地领取。

### 9.2 医疗保险关系转移
1. 职工跨统筹地区就业的，其基本医疗保险关系随本人转移，缴费年限累计计算。
2. 个人账户余额可随关系转移一并划转。
3. 可在新参保地医保经办机构申请转移接续。

### 9.3 失业保险关系转移
- 职工跨统筹地区就业的，失业保险关系随本人转移，缴费年限累计计算。
- 失业保险待遇按转入地标准执行。

## 十、社会保险费率与缴费实务

### 10.1 五险总费率参考
各类社会保险合计缴费比例（单位和个人合计）通常为工资总额的**35%-42%**左右，各地略有差异：

| 险种 | 单位缴费 | 个人缴费 | 合计 |
|------|---------|---------|------|
| 养老保险 | 16% | 8% | 24% |
| 医疗保险 | 6%-10% | 2% | 8%-12% |
| 失业保险 | 0.5%-1% | 0.2%-0.5% | 0.7%-1.5% |
| 工伤保险 | 0.2%-1.9%（按行业风险） | 0 | 0.2%-1.9% |
| 生育保险 | 0.5%-1%（已并入医保） | 0 | 0.5%-1% |

### 10.2 缴费基数上下限
- **上限**：当地上年度职工平均工资的300%
- **下限**：当地上年度职工平均工资的60%
- 各地每年7月左右公布当年社保缴费基数上下限（根据上年度社平工资调整）

### 10.3 用人单位常见违法情形
1. **不缴或少缴**：不为员工参保或以最低基数参保；
2. **协议免除**：与劳动者约定"自愿放弃社保"（此类约定无效，缴纳社保是法定义务）；
3. **试用期不缴**：以"试用期"为由不为员工参保（违法，用工之日起30日内须参保）；
4. **以补贴代替**：向员工发放"社保补贴"代替实际参保（仍属违法）。

## 十一、个人养老金制度

### 11.1 制度概述
根据国务院办公厅《关于推动个人养老金发展的意见》（国办发〔2022〕7号），个人养老金是政府政策支持、个人自愿参加、市场化运营的补充养老保险制度，属于养老保险**第三支柱**。

### 11.2 参加条件
- 在中国境内参加城镇职工基本养老保险或者城乡居民基本养老保险的劳动者，均可自愿参加个人养老金。
- 每年缴纳上限为**12000元**（根据经济社会发展水平适时调整）。

### 11.3 账户管理
- 参加人通过个人养老金信息管理服务平台，建立**个人养老金账户**；
- 在商业银行开立个人养老金资金账户，用于缴费、购买产品、归集收益等。

### 11.4 税收优惠
- 缴费环节：按12000元/年的限额标准，在综合所得或经营所得中据实扣除；
- 投资环节：计入个人养老金资金账户的投资收益暂不征收个人所得税；
- 领取环节：领取的个人养老金不并入综合所得，单独按照**3%**的税率计算缴纳个人所得税。

### 11.5 领取条件
- 达到领取基本养老金年龄；
- 完全丧失劳动能力；
- 出国（境）定居；
- 国家规定的其他情形。
- 领取方式：可按月、分次或者一次性领取。

### 11.6 投资产品范围
个人养老金资金账户资金可用于购买符合规定的：
- 银行理财
- 储蓄存款
- 商业养老保险
- 公募基金（养老目标基金等）

## 十二、工伤保险伤残等级赔偿明细

### 12.1 一次性伤残补助金

根据《工伤保险条例》，工伤职工经劳动能力鉴定构成伤残等级的，按以下标准从工伤保险基金支付一次性伤残补助金（以本人工资为基数）：

| 伤残等级 | 补助金标准 |
|---------|-----------|
| 一级 | 27个月本人工资 |
| 二级 | 25个月本人工资 |
| 三级 | 23个月本人工资 |
| 四级 | 21个月本人工资 |
| 五级 | 18个月本人工资 |
| 六级 | 16个月本人工资 |
| 七级 | 13个月本人工资 |
| 八级 | 11个月本人工资 |
| 九级 | 9个月本人工资 |
| 十级 | 7个月本人工资 |

**注意**："本人工资"指工伤职工因工作遭受事故伤害或者患职业病前12个月平均月缴费工资。本人工资高于统筹地区职工平均工资300%的，按300%计算；低于统筹地区职工平均工资60%的，按60%计算。

### 12.2 伤残津贴（按月支付）

| 等级 | 津贴标准 | 支付来源 |
|------|--------|---------|
| 一级 | 本人工资的90% | 工伤保险基金 |
| 二级 | 本人工资的85% | 工伤保险基金 |
| 三级 | 本人工资的80% | 工伤保险基金 |
| 四级 | 本人工资的75% | 工伤保险基金 |
| 五级 | 本人工资的70%（保留劳动关系，安排适当工作；难以安排工作的发伤残津贴） | 用人单位 |
| 六级 | 本人工资的60%（同上） | 用人单位 |

- 1-4级伤残的伤残津贴由工伤保险基金按月支付，实际低于当地最低工资标准的，由工伤保险基金补足差额。
- 5-6级伤残的伤残津贴由用人单位按月支付，实际低于当地最低工资标准的，由用人单位补足差额。
- 7-10级伤残不享受伤残津贴。

### 12.3 一次性工伤医疗补助金和伤残就业补助金

5-10级伤残职工，经职工本人提出与用人单位解除或者终止劳动关系时（劳动、聘用合同期满终止或职工本人提出解除的），由工伤保险基金支付**一次性工伤医疗补助金**，由用人单位支付**一次性伤残就业补助金**。具体标准由各省级人民政府规定（因各省标准差异较大，此处不做统一列举）。

### 12.4 停工留薪期待遇

1. **停工留薪期**：职工因工作遭受事故伤害或患职业病需要暂停工作接受工伤医疗的期间。
2. **工资待遇**：停工留薪期内，原工资福利待遇不变，由所在单位按月支付。
3. **期限**：一般不超过12个月。伤情严重或情况特殊，经设区的市级劳动能力鉴定委员会确认，可以适当延长，但延长不得超过12个月（最长24个月）。
4. **生活护理费**：评定伤残等级后需要生活护理的，从工伤保险基金按月支付生活护理费，标准为：
   - 生活完全不能自理：统筹地区上年度职工月平均工资的50%
   - 生活大部分不能自理：40%
   - 生活部分不能自理：30%

### 12.5 工亡待遇

职工因工死亡的，其近亲属可领取：
1. **丧葬补助金**：6个月的统筹地区上年度职工月平均工资
2. **供养亲属抚恤金**：按职工本人工资的一定比例发给无劳动能力的亲属，配偶每月40%，其他亲属每人每月30%，孤寡老人或孤儿在此基础上增加10%，总额不超过职工生前工资的100%
3. **一次性工亡补助金**：上一年度全国城镇居民人均可支配收入的20倍

## 十三、视同工伤情形

### 13.1 视同工伤的情形
根据《工伤保险条例》第十五条，职工有下列情形之一的，**视同工伤**，享受工伤保险待遇：
1. 在工作时间和工作岗位，**突发疾病死亡**或者在**48小时之内经抢救无效死亡**的；
2. 在抢险救灾等维护国家利益、公共利益活动中受到伤害的；
3. 职工原在军队服役，因战、因公负伤致残，已取得革命伤残军人证，到用人单位后旧伤复发的。

### 13.2 不得认定为工伤的情形
职工虽符合应当认定工伤或视同工伤的条件，但有下列情形之一的，**不得认定为工伤或者视同工伤**：
1. **故意犯罪**的；
2. **醉酒或者吸毒**的；
3. **自残或者自杀**的；
4. 法律、行政法规规定的其他情形。

### 13.3 上下班途中工伤认定的细化
"在上下班途中，受到非本人主要责任的交通事故或者城市轨道交通、客运轮渡、火车事故伤害的"认定为工伤，需要注意：
1. "上下班途中"包括在合理时间内往返于工作地与住所地、经常居住地、单位宿舍的合理路线的途中；
2. "非本人主要责任"的认定以公安机关交通管理部门出具的事故责任认定书为准；
3. 如果是本人主要责任或全部责任的，**不认定为工伤**。
